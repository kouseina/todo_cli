# Todo (CLI)

> Command-line tool to manage the Todo lists

# Create a Todo App

## Requirement Commands

* Add a todo

    Ex:  
    `add` Go to market  
    `add` Buy a Food  

* List all todo

    Ex:  
    `ls`  
    ```
    (1) Go to market  
    (2) Buy a Food
    ```

* Remove a todo

    Ex:  
    `rm` 1  
    `ls`
    ```
    (1) Buy a Food
    ```

* Check a todo

    Ex:  
    `check` 1  
    `ls`
    ```
    (1) Go to market ✅  
    (2) Buy a Food
    ```

* Uncheck a todo

    Ex:  
    `uncheck` 1  
    `ls`
    ```
    (1) Go to market  
    (2) Buy a Food
    ```
    
* Toggle a todo

    Ex:  
    `toggle` 1  
    `ls`
    ```
    (1) Go to market ✅
    (2) Buy a Food
    ```
